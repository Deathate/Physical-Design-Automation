# Lab1 - C++ Program README

## Introduction
This repository contains the source code and a Makefile for a C++ program called Lab1. Lab1 is designed to implement Fiduccia-Mattheyses algorithm.

## Prerequisites
Before you can build and run Lab1, make sure you have the following installed on your system:
- [C++ Compiler]: You need a C++ compiler to build the program.
- [Make]: The Makefile is used to automate the build process. Make sure it's installed on your system.

## Build Instructions
Follow these steps to build the Lab1 program:

1. Open a terminal or command prompt.

2. Navigate to the directory where this README.txt file is located.

3. Run the following command to build the Lab1 program:
    make
    ./Lab1 [input file name] [output file name]
